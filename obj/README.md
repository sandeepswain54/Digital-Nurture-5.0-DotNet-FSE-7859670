# Week 1 - SQL Exercise (Advanced Concepts)

This folder contains hands-on SQL exercises completed as part of the Digital Nurture 5.0 .NET FSE program.

## Contents

| Folder | Exercise |
|---|---|
| 01_Ranking_and_Window_Functions | ROW_NUMBER, RANK, DENSE_RANK, PARTITION BY |
| 02_SQL_Exercise_Index | Non-clustered, clustered, and composite indexes |
| 04_SQL_Exercise_Stored_procedure | Creating and executing stored procedures |

## How to run

- Exercise 1 can be run on any SQLite-based SQL editor (e.g. sqliteonline.com).
- Exercises 2 and 4 require SQL Server syntax and were run on dbfiddle.uk with the database set to SQL Server.
